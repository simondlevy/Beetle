<h1>Software for <i>Bracketing the Beetle</i></h2>

This <a href="wittvsa.zip">zip file</a> contains Matlab code demonstrating some 
ideas in a 
<a href="http://home.wlu.edu/~levys/pubs/agi_2014_levy.pdf">paper</a> 
presenting 
<a href="http://home.wlu.edu/~levys/vsa.html">Vector Symbolic Architectures</a> (VSA) as a solution to some of the
issues raised in the later work of
<a href="http://en.wikipedia.org/wiki/Wittgenstein">Ludwig Wittgenstein</a>.

<p>

To try out these demos, download and unzip the file, launch Matlab, and
do <b>Set Path / Add with Subfolders ...</b>
to add the <b>wittvsa</b> folder and its subfolders to your path. You can then
run the following demos:

<ul>
<li> <b>ravens</b> Raven's Progressive Matrices demo
<li> <b>necker</b> Necker cube demo
<li> <b>sdmsignoise</b> Sparse Distributed Memory signal-to-noise demo 
<li> <b>sdmsequence</b> Sparse Distributed Memory sequence demo

</ul>
